## What is Parts Catalog?

Parts Catalog is a comprehensive CRUD web application designed for companies to manage their automotive parts inventory. Built with Spring Boot and Vue.js, it provides a robust platform for tracking parts, vehicles, and their relationships.

## Key Features

<CardGroup cols={2}>
  <Card title="Parts Management" icon="gear" href="/features/parts-management">
    Complete CRUD operations for managing parts with advanced search capabilities
  </Card>
  <Card title="Vehicle Tracking" icon="car" href="/features/vehicles-management">
    Organize and manage vehicle information with alphabetical listing
  </Card>
  <Card title="Relationships" icon="link" href="/features/part-vehicle-relationships">

## Technology Stack

<CardGroup cols={3}>
  <Card title="Backend" icon="server">

  </Card>
  <Card title="Frontend" icon="browser">
